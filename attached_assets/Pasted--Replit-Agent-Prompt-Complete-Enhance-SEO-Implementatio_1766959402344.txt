## 2. Implement Complete Schema Markup (JSON-LD)

**Task**: Add three types of structured data to the homepage using proper JSON-LD format.

### Implementation Method:

Create a new component: `components/StructuredData.tsx`

```typescript
import Script from 'next/script'

export default function StructuredData({ data }: { data: any }) {
  return (
    <Script
      id={`structured-data-${data['@type']}`}
      type="application/ld+json"
      dangerouslySetInnerHTML={{ __html: JSON.stringify(data) }}
      strategy="beforeInteractive"
    />
  )
}
```
