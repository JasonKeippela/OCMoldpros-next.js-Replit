## 301 redirect requirements

If any old route differs from the selected canonical route:

* create a **301 permanent redirect**
* make every redirect single-hop only
* avoid chains like old → intermediate → final
* preserve important legacy SEO pages
* do not redirect unrelated location pages unless they clearly correspond to a service page
* do not redirect blindly; verify route intent before mapping

Implement redirects in `next.config.js` or `next.config.ts` like this:

```ts
async redirects() {
  return [
    {
      source: '/old-service-url',
      destination: '/home-inspector/services/mold-sampling',
      permanent: true,
    },
  ];
}
```
