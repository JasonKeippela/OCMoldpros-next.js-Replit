# Next.js Schema Markup Implementation for OC Mold Pros

## Instructions for Replit Agent

Please implement the following schema.org structured data markup for ocmoldpros.com. The site is built on Next.js and needs LocalBusiness, Service, Review, and FAQPage schema added.

## Implementation Requirements

1. Create a new component at `components/StructuredData.tsx` that exports schema JSON-LD
2. Import and render this component in the main layout or home page
3. Use Next.js Script component with type="application/ld+json"
4. The schema should be valid JSON-LD format

## Component Structure

Create the component with this structure:

```typescript
import Script from 'next/script';

export default function StructuredData() {
  const localBusinessSchema = { /* LocalBusiness JSON */ };
  const reviews = [/* Array of review schemas */];
  const faqSchema = { /* FAQ JSON */ };

  return (
    <>
      <Script
        id="local-business-schema"
        type="application/ld+json"
        dangerouslySetInnerHTML={{ __html: JSON.stringify(localBusinessSchema) }}
      />
      <Script
        id="reviews-schema"
        type="application/ld+json"
        dangerouslySetInnerHTML={{ __html: JSON.stringify(reviews) }}
      />
      <Script
        id="faq-schema"
        type="application/ld+json"
        dangerouslySetInnerHTML={{ __html: JSON.stringify(faqSchema) }}
      />
    </>
  );
}
```

## Where to Add

Import and use `<StructuredData />` in:
- `app/layout.tsx` (for site-wide schema), or
- `app/page.tsx` (for homepage only)
